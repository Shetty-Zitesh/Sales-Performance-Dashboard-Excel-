# Sales Performance Dashboard (Excel)

## Project Overview
This project is an interactive Sales Performance Dashboard built using Excel 2016 to analyze sales trends, target achievement, and customer performance.

## Key Features
- Data cleaning and transformation using Power Query
- Lookup logic using VLOOKUP/XLOOKUP
- KPI metrics: Total Sales, Target, Achievement %, Gap
- Pivot Tables and Pivot Charts
- Interactive slicers (Date, Region, Category)
- Sales analysis by Region, Category, and Customer

## Tools Used
- Microsoft Excel 2016
- Power Query
- Pivot Tables & Pivot Charts
- Advanced Excel formulas

## Business Insights
- Identified underperforming regions vs targets
- Analyzed top-performing customers
- Measured sales achievement against regional targets
